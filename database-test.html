Database Test Console

Click the "Run Database Tests" button in the bottom right corner to test the database functionality.

This test will:

  • Initialize the database
  • Save a test entry with an image
  • Retrieve all entries
  • Retrieve a specific entry
  • Retrieve images for an entry
  • Export all data
  • Delete an entry

Results will be displayed in a popup window.
